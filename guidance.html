INSTRUCTIONS

  1. Start the Game: Once you click the START GAME button the game will start.
  2. Jump Over Obstacles: As the game starts, your dinosaur will begin to run automatically from left to right. You need to press the Space bar key to make the dinosaur jump over obstacles like cactus.
  3. Avoid Collision: The goal is to avoid colliding with the obstacles. If you hit an obstacle, the game is over, and your score is displayed.
  4. Collect Points: Your score is determined by how far you can run without hitting an obstacle. The game doesn't have an end, but it keeps getting faster and more challenging the longer you play.
  5. Challenge Yourself: The game can be quite addictive, and you can challenge yourself to beat your high score.
